Canvas

O elemento <canvas> é utilizado para se desenhar elementos graficos em uma pagina com JavaScrispt

O elemento em si é apensas um contêiner para os desenhos criados. Podemos fazer varias operações de desenho com javaScript, como traçar uma linha, criar caixas, circulos, texto e imagens

Podemos utilizar para a criação de graficos, animações com interações do ususário, etc.

Por possibilitar animações e interações com o usuario, é muito utilizado para o desenvolvimento de jogos

Nosso primeiro canvas

Para se criar um canvas basta declararmos a tag, indicar as dimensões e um ID, que utilizaremos para acessálo pelo Java Script

<canvas id="meuCanvas" width="400" height="300"></canvas>

Agora vamos pegar o elemento e jogalo em uma variavel

var meuCanvas = document.getElementById('meuCanvas')

var ctx = meuCanvas.getcontext("2d")

Ao pegar o elemento e guarda-lo numa variavel, pegamos o context do nosso elemento e indicamos que utilizaremos as funcionalidades "2D". Guardamos o contexto na varivel "ctx". É a partir dessa variavel que faremos todas as operações no nosso canvas

feito isso, abra o console do devtools e execute os seguintes codigos para ver o que da para fazer

Desenhando linhas

ctx.moveTo(5, 5)

ctx.lineTo(100, 300)

ctx.stroke()

Desenhando Circulos

ctx.arc(200,150,60,0,2*Math.PI)

ctx.stroke()
