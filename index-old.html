Minha vida profissional foi recheada de desafios. Com a experiencia adquirida nesse projeto fui direcionado para participar da migração dos sistemas desnvolvidos em Cobol para SQL usando o Oracle, onde aprendi muito de SQL, cometi erros, aprendi com eles e assim comecei a ganhar mais experiencia.

Viajei durante 2 anos acompanhando os representantes para adquirir o conhecimento necessário para passar as informações para a area de TI desenvolver o sistema de vendas.
