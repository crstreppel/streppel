    Ao reduzir ou abrir a pagina em um celular o código será exibido com fundo amarelo e letras pretas

    Isso foi feito incluindo a média query no css:
    @media print {
    li[class]::after {
    content: " - " attr(class);
    }

    Para podermos fazer com que nosso CSS exiba o nosso HTML de uma certa maneira em cada dispositivo, foram criados os Media Types. Por eles nós podemos indicar para que tipo de mídia o nosso CSS vai funcionar.


    Aqui temos uma lista dos tipos de mídia disponíveis:

  • all: Para todos os dispositivos;
  • print: Para impressão ou para documentos na tela de prévia para impressão;
  • screen: Para dispositivos com telas coloridas;
  • speech: Para sintetizadores de voz.
